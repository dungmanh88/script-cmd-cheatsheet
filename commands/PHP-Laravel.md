# Change laravel version
```
Change in composer.json

"require": {
    "php": ">=5.6.4",
    "laravel/framework": "5.3.*",
    "laravelcollective/html": "5.3.*",
    "barryvdh/laravel-elfinder": "^0.3.8",
    "magyarjeti/laravel-lipsum": "^1.1"
}
```

# Get laravel version
```
php artisan --version
```
